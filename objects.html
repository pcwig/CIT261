Creating Objects

In order to breed an object in javascript you must declare the object. To declare an object simply use the keyword var followed by the object name.

var objectname

Sometimes we like to have an "object type" that can be used to create many objects of one type.The standard way to create an "object type" is to use an object constructor function:

function dog(breed, age, weight, color) {
  this.breed = breed;
  this.age = age;
  this.weight = weight;
  this.color = color;
  }

You can create an object using the the keyword new using that constructor: 

var objectname = new FunctionName
var kevin = new dog())

Example

In this example, we will use a constructor function to allow us to create a new dog object. We will take user inputs to create properties and then use those properties to display some generated text.


function createDog() {

//Retreive inputs from user                    
var dogBreed = document.getElementById("dogBreed").value;
var dogAge = document.getElementById("dogAge").value;
var dogWeight = document.getElementById("dogWeight").value;
var dogColor = document.getElementById("dogColor").value;

//Create a constructor function for dog					 
function dog(breed, age, weight, color) {
	this.breed = breed;
	this.age = age;
	this.weight = weight;
	this.color = color;
}

//creat a myDog object
var myDog = new dog(dogBreed, dogAge, dogWeight, dogColor);

//output a string using properties					
document.getElementById("dogOutput").innerHTML = "I own a " + myDog.color +
 " " + myDog.breed + " that is " + myDog.age + " years old."
+ " and weighs " + myDog.weight + " pounds.";
}
