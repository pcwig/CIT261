JSON

JSON Basics

  • JSON stands for JavaScript Object Notation.
  • Lightweight text-based open standard designed for human-readable data interchange.
  • Used in addition for serializing and transmitting structured data over network connections.
  • Meant to be easy for humans to read and write

    • Datatypes

  • Number: Integer and Float stores the same
  • String: Uses double quotes to distinguish
  • Boolean: True or False
  • Array: Ordered list of 0 or more values
  • Object: Unordered collection of key/value pairs

    • JSON Syntax

  • JSON uses key/value Pairs 
    • {“name”:”John”}
  • Must use the specified data types
  • Uses double quotes around KEY and VALUE
  • File type is “.json”

    • Example of a JSON File

    Here is a simple JavaScript object

    var person1 = {
    firstName: "John",
    lastName: "Smith",
    age : 31,
    livesInUS: true,
}

    This is what it would look like if converted to JSON

    {
    "firstName": "John",
    "lastName: "Smith",
    "age" : "31",
    "livesInUS": "true",
}

    Stringify

    Where JSON can really help you, is by taking objects and turning them into a JSON string in order to send them to somewhere else.

    In order to do this us must use the JSON.stringify code

    Using the "person1" example, we can store the JSON text into a variable like so:

    var string = JSON.stringify(person1);

    Parsing

    Parsing allows you to take the JSON file or string and put it back into a variable in order to use within your program

    In order to do this us must use the JSON.parse code

    Using the string variable we used in the last example we can put the JSON string back into a JavaScript Variable and then call certain parts of that variable as so:

    //Take the string variable and change it back into an object with the parse command
var parse = JSON.parse(string);
//print out the first name and age
document.getElementById("output").innerHTML = parse.firstName + ", " + parse.age ;
