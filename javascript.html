JavaScript

Variables

Variables are containers for storing data values. A variable is a location within the computer's memory to store data When we declare a variable the computer will reserve space within its memory for that variable.

This is how you declare a variable.

var idNumber;
var name;
var taxRate;

You store a value within the variable by using the equals sign (=).

var idNumber = 445696;
var name = "Jim";
var taxRate = .06;

Loops

Every loop includes a boolean condition. Each time the computer needs to decide if it should repeat the statements inside a loop, the computer evaluates the loop's condition. If the condition evaluates to true, then computer repeats the statements inside the loop. If the condition evaluates as false, the computer does not repeat the inside statements but instead executing the program starting with the statements after the loop.

There are four basic loops in JavaScript. They are:

While - Used for simple counting, a sentinel needs to be checked before the loop.

For - Used for simple counting loop.

For Each - Causes statements inside of it to be repeated once for each element in an array or object.

Do While - Used for a simple counting loop, the statements need to be checked before the sentinel.

While

var i = 1;
while (i < 3) {
    window.alert(i);
    i++;
}

For

for (var i = 1; i < 3; i++) {
    window.alert(i);
}

For Each

var text = "";
var i;
for (i = 0; i < 5; i++) {
    text += "The number is " + i + "
";
}

document.getElementById("demo").innerHTML = text;

Do While

var i = 1;
do {
    window.alert(i);
    i++;
}
while (i < 3);

Conditional Statements

A computer must be able to make decisions. The computer makes decisions by evaluating a condition and if the condition is true, then certain statements will be executed. If the condition is false, then the computer selects a different set of statements to execute.

Here are the following Conditional statements:

If - Used when a true choice exists and the false choice does not exist.

If...Else... - Used when a true and false choice exists.

If...Else If...Else - Used when more than two choices exist. Each choice is based on different variables.

Switch - Used when more than two choices exist. Each choice is based on the same variable and uses equals (==). Comparison Values are integers or string.

Switch

var registerDate;
switch(classStanding) {
    case 'F':
        registerDate = "Nov 23";

        registerDate = "Nov 17";

Arrays

An array is a collection of variables where each variable has the same data type. Each variable in an array is called and element and has an index, which tells you the location of data in the array. A big advantage of using an array is that all the elements within the array can be declared, created, or passed to a function as a whole.

Declaring an Array - var arrayName;

Creating an Array - arrayName = new Array(size);

You can initialize and array at the same time you declare and create it - var arrayName = [value0, value1, ... valueN];

var scores = new Array(25);

//or

var scores = [10, 12, 13, 14, 15, 16, 17, 18];

Associative Arrays

Arrays with named indexes are called associative arrays.

var person = [];
person[0] = "James";
person[1] = "Kenny";
person[2] = "Sarah";
document.getElementById("example0").innerHTML = person[0];
document.getElementById("example1").innerHTML = person[1];
document.getElementById("example2").innerHTML = person[2];

Function Parameters

Function parameters are the names listed in the function definition. Function arguments are the real values passed to, and received by the function.

function average(a, b, c) {
    var sum = a + b + c;
    var avg = sum / 3;
    return avg;
}
