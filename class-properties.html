Manipulating CSS Class Properties Using JavaScript:

Manipulating CSS with JavaScript is a fairly simple process.

To change the style of an HTML element, use this syntax:

document.getElementById(id).style.property = new style

Changing the Background Color

function getColor(color) {
    document.body.style.background = color;
}

Changing a Class Color

function getClassColor(color) {
    var text = document.getElementsByClassName('colorClass');
    text[0].style.background = color;
}

Click the button to change the class color.

Adding and removing a class to a div

Another way to manipulate CSS with JavaScript is to have CSS code setup for a certain class and then adding that class to an element. That element will then take on the style of that class.




Test Area

Code:

function addClass1() {
    document.getElementById("class").classList.add("box");
}

function addClass2() {
    document.getElementById("class").classList.add("class2");
}

function addClass3() {
    document.getElementById("class").classList.add("thirdClass");
}

function removeClass1() {
    document.getElementById("class").classList.remove("box");
}

function removeClass2() {
    document.getElementById("class").classList.remove("class2");
}

function removeClass3() {
    document.getElementById("class").classList.remove("thirdClass");
}

CSS:

.box {
    width: 400px;
    height: 50px;
    padding: 15px;
    border: 1px solid black;
}

.class2 {
    background-color: coral;
    color: white;
}

.thirdClass {
    text-transform: uppercase;
    text-align: center;
    font-size: 25px;
}
