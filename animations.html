CSS3 Animations

Keyframes and Animations work together to make an animated element.

Keyframes are what the animation does such as where it moves or how much it expands.

Animation is how it's done. Think speed, timing, duration etc.

Keyframes

The @keyframe rule allows you to make changes in the animation code. These changes set the CSS style to whatever you want within the animation time frame. this can be done multiple times at whatever points you want.

As an example, if you wanted to change the width of an element you it could look something like this.

@keyframes widthChange {
    from {width:100px;}
    to {width:200px;}
    }
    
	OR
	
@keyframes widthChange {
    0% {width:100px;}
    100% {width:200px;}	
    }

Example 1

The 'to' and 'from' represent 0% through the animation time and 100% through the animation time. These percentages represent the time within the alloted animation time when things will change.

You may add more changes by adding additional percentages like so.

@keyframes changeWidth {
		0% {width: 100px;}
		25%  {width: 600px;}
		75%  {width: 50px}
		100% {width: 100px;}
	}

Animation

The animation propery has eight different values it can use to change how the animation reacts.


Value Description
animation-name Specifies the name of the keyframe you want to bind to the selector
animation-duration Specifies how many seconds or milliseconds an animation takes to complete
animation-timing-function Specifies the speed curve of the animation
animation-delay Specifies a delay before the animation will start
animation-iteration-count Specifies how many times an animation should be played
animation-direction Specifies whether or not the animation should play in reverse on alternate cycles
animation-fill-mode Specifies what values are applied by the animation outside the time it is executing
animation-play-state> Specifies whether the animation is running or paused

These values can be written out individually or in shorthand as so.


animation-name: test;
animation-duration: 0s;
animation-timing-function: ease;
animation-delay: 0s;
animation-iteration-count: 1;
animation-direction: normal;
animation-fill-mode: none;
animation-play-state: running;

OR 

animation: test 0s ease 0s 1 normal none running;}

Each of these values has a default property so if you don't chose something it will fill in the default value.

Putting it together

Here is full code from Example 1 with the animation property.

.example2 {
	width: 100px;
	height: 100px;
	background: blue;
	position :relative;
	animation: changewidth2 10s infinite;
	}
@keyframes changewidth2 {
	0% {width: 100px;}
	25%  {width: 600px;}
	75%  {width: 50px;}
	100% {width: 100px;}

And another one slightly more complex

.example3 {
    font-size:100px;
    animation-name: slideAround;
    animation-duration: 5s;
    animation-timing-function: linear;
    animation-delay: 2s;
    animation-iteration-count: infinite;
    animation-direction: alternate;
    }
	 
@keyframes slideAround {
    0% {
        color: red; 
        left: 0px; 
        top: 0px;
    }
	
    25%  {
        color: yellow;
        left: 200px; 
        top: 0px;
    }
		
    50%  {
        text-color: blue; 
        left: 200px; 
        top: 200px;
    }
		
    75%  {
        color: green; 
        left: 0px; 
        top: 200px;
    }
    100% {
        color: red; 
        left: 0px; 
        top: 0px;
    }
