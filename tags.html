HTML5 Tags - Video, Audio, and Canvas

HTML5 offers many new and improved tags for media such as Video, Audio and Canvas tags. This page will demonstrate each of these and explore some of their capabilities.

Video

The video tag allows you to embed video into your site by specifying a source and a type for the video.

The video tag allows you to support multiple video formats that the browser will cascade through until a compatible one is found for the current device. It no format is compatible with the device, you are able to fall back on an error message.


    <video width="100%" height="auto" controls>
    <source src="video/toyboat.mp4" type="video/mp4">
    <source src="video/toyboat.mp4" type="video/ogg">
    Your browser does not support the video tag.
</video>

Canvas

The HTML canvas tag is used to draw graphics on a web page.

The graphic underneath was used to create an animation.

The canvas element is only a container for graphics. You must use JavaScript to actually draw the graphics.

JavaScript

function draw(x,y){
    var canvas = document.getElementById('canvas');
    var ctx = canvas.getContext('2d');
    ctx.save();
    ctx.clearRect(0,0,550,100);
    ctx.fillStyle = "rgba(50,200,200,1)";
    ctx.fillRect (x, 10, 75, 75);
    ctx.restore();
    x += 2;
    var loopTimer = setTimeout('draw('+x+','+y+')',20);
}
